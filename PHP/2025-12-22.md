## 学んだこと ①

### 配列について

- 基本は`$配列名=[値1,値2,・・・]`という形で書き、インデックス番号が自動的に 0 から振られる。
- アクセスする場合は`＄配列[インデックス番号]`で記入する。
- 文字列キーを用いた配列を連想配列といい、`$配列名=[キー1=値1,キー2=値2,・・・]`と書く。
- さらに多次元に配列することもできる。例を下に示す。

```php
$member = [
    "name" => "田中太郎",
    "gender" => 1,
    "address" => "徳島県徳島市",
    "tel" => "080 - 888 - 888",
    "email" => "aaa@aaa.com",
    "hobbies" => [1 => "散歩", 2 => "読書"]
];
```

## 学んだこと ②

### foreach 命令について

- 指定された配列/連想配列の要素を取り出して、先頭から順番に処理します。
- 構文は`foreach(配列 as 値変数){//ループ内で実行する処理}`または、`foreach(配列 as キー変数 => 値変数){//ループ内で実行する処理}`である。
