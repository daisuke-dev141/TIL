## 学んだこと②
### 明示的な変換について

- PHPではその時の状況に応じて適切なデータ型へ変換される、これを暗黙的な変換という。
- `(int)値`で整数型へのキャストが行われる。
- `var_dump`は変数の情報を出力してくれる関数である。


#### 例
```php
<?php
var_dump((int)1540.95);    //結果:int(1530)
```

## 学んだこと③
### 演算子について

- 代数演算子は四則演算などで`'108'+'1.5XYZ'`は結果、数値リテラルとみなされ`109.5`になるが、エラーが出る。
- 代入演算子は左辺で指定した変数に対して、右辺の値をセットするための演算子で`$x=5; $x+=2;`の結果は`7`になる。
- 比較演算子は左辺と右辺を比較し、その結果をtrue/falseとして返す演算子で`var_dump('13xyz'==13);`の結果はPHP8から`false`となった。
  条件演算子は`条件式?式1:式2`であり条件式がtrueの場合は式1、条件式がfalseの場合は式2となる。
- 論理演算子は複数の条件式(または真偽値)を論理的に結合し、その結果をtrue/falseで返す演算子で`$x && $y`の結果は`false`である。
- 文字列演算子(.)は左式/右式の文字列を連結する演算子であり下記のようになる。

#### 例(文字列演算子)
```php
<?php
$x = 'こんにちは、';
$y = '世界！';
print $x . $y;      //結果 : こんにちは、世界!
```

## 学んだこと③
### 条件分岐(if命令)について

 - 構造化プログラミング(順次/選択/反復)の選択である。
 - 閏年かどうかの条件分岐をif文で書いたものが下記です。

#### 例(文字列演算子)
```php
<?php
    $year = 100;
    if ($year % 400 === 0) {
        print '閏年です';
    } elseif ($year % 100 === 0) {
        print '閏年ではありません';
    } elseif ($year % 4 === 0) {
        print '閏年です';
    } else {
        print '閏年ではありません';
    }
```
