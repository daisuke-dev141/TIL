## 学んだこと ②

### 繰り返し処理について

- 構造化プログラミングの「反復」です。
- for 命令はあらかじめ指定された回数だけ処理する命令です。`for(初期化式; 継続条件式; 増減式){//ループ内で実行する処理}`
- while 命令は与えられた条件が true である間処理を繰り返します。`while(条件式){//条件式がtrueである間、繰り返し実行する処理}`、データベースとのやりとりで使うことが多い
- do~while 命令は、条件はなんであれとりあえず 1 回目は実行される。while は条件見てから。while の後に`;`をつけることに注意
- 強制的にループを中断したい場合に用いる命令もある。break 命令は、現在のループを強制的に中断する命令である。continue 命令は現在の周回だけをスキップし、ループそのものは継続して実行する命令である。
- 制御命令は互いに入れ子(ネスト)にできます。その例を下に記述します。
#### 例

```php
<?php
echo "<h1>9×9表　2次元ループの表</h1>";

    echo '<table border = 1>';
    for ($i = 1; $i <= 9; $i++) {
        //縦列
        echo '<tr>';
        for ($j = 1; $j <= 9; $j++) {
            //横列
            echo '<td width = 50px>', $i * $j . "</td>";
        }
        echo '</tr>';
    }
    echo '</table>';
    ?>
