## 学んだこと ①

### 日付関数

- date()関数はタイムスタンプ値を指定フォーマットで成形
- time()関数は現在のタイムスタンプ値を取得
- strtotime()関数は与えられた日付文字列に対応するタイムスタンプ値を取得

```php

//1日の秒数
const DAY_SECONDS = 24 * 60 * 60;

//今の時間
$postDate = date("Y-m-d H:i:s");
//講座終了日のタイムスタンプ
$end_time = strtotime("2026-3-18");
//ただ今のタイムスタンプ
$now_time = time();

//差分
$diff = $end_time - $now_time;
$days = ceil($diff / DAY_SECONDS);
echo "講座終了まで{$days}日";

```

## 学んだこと ②

### ユーザー定義関数

- 標準的な関数ではカバーされないような定型的な処理についてユーザー自ら定義する関数
- `function 関数名(仮引数,…){//任意の処理 return 戻り値;}`
- 関数名は`getTriangleArea`のように camelCase 記法で書く
- スクリプトの外部化は`require__DIR__. 'ファイルの相対パス'`;で記入する
- 関数の中がローカル、それを含めた全部がグローバル
