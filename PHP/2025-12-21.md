# Docker学習メモ（PHP開発環境）

## 学習目的
今後Laravelを独学でやるにあたって学習用にxxamp以外の開発環境を構築するため  
Dockerを使って Apache + PHP 環境を構築した。

---

## 構成

```text
docker/
└─ php-study/
   ├─ docker-compose.yml
   └─ src/
      └─ index.php
```


---

## 使用技術
- Docker
- Docker Compose
- PHP 8.2
- Apache
- VSCode

---

## docker-compose.yml の内容
```yml
version: "3.8"

services:
  php:
    image: php:8.2-apache
    ports:
      - "8080:80"
    volumes:
      - ./src:/var/www/html


##起動からブラウザ確認までの流れ
①Docker Desktopを起動させる
②`cd php-study`でフォルダを指定
③`docker compose up -d`でdockerを起動
④`http://localhost:8080/ファイル名`でブラウザに表示する
